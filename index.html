Quickstart SVN to GIT

This is mostly taken from http://git-scm.com/course/svn.html This covers command line operations and their equivalents.
This may differ based on UI choice. I personally use git with command line and prefer it over GUIs.

Starting a local repository from scratch or existing project

git init svnadmin create repo
git clone repo svn checkout repo

Adding a file and committing

git add filename svn import file://repo
git commit
git push		 svn commit
Committing in git and svn are different. A commit in git only pushes the change to your local repository, which you already have anyway. A push shares your commit to whatever remote branch you have tracking your branch. We will get into more details about branching in a little bit.

diff, ignore

git diff svn diff | less
